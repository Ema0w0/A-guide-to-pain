# Description
A simple Kill Aura script that when the button is toggled will activate and create an area around you in which if another player walks into, will kill them even if your not looking directly at them. 
Works on executors Yub-X, Potassium, Delta, Ronix PC, Wave, Volt, Ronix Mobile, Electron and more!
